Supermarket simulator

See project on Github

This project simulates customers shopping in a supermarket using Java. The aim was to make it easy for key staff within an organization to represent the components of a supermarket scenario and run a simulation based on various configurations in order to maximise the efficiency of a store.

The lessons I learned while building this project were as follows: what a design pattern is, the different types of design pattern, which one to choose and how to efficiently use it to integrate it to any projects.

I also learned How to implement a project in a multi-threaded environment.
