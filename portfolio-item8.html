Countries app

Github try the demo

Small ReactJS app that displays general information about certain such as the country name, the capital and the population. This app also allows the user the filter countries by continent, and to decide the number of countries displayed on the screen.

This app is my second individual ReactJS project. By making this project I learnt the difference between state and stateless components in React, and I decided to make stateless components to produce this app. Moreover, I understood how to use the "useEffect()" hook more efficiently. Furthermore, this app allow me to use a Rest Api called "countriesApi". Using an Api in React was a first time for me, and I learnt a lot on how to use an Api efficiently in React.
